# react-leaflet-demo
Displaying maps in a react application

## Setup / Run

| command script  | description  | notes |
|---|---|---|
| `yarn`  | Installs/refreshes node package dependencies |---|
| `yarn start`  |  Starts react application | runs locally on port 3000 |
